Ты можешь сделать простой сайт для меня?
ChatGPT сказал:

Конечно! Могу помочь создать простой сайт. Вот основные шаги, которые понадобятся для создания базовой страницы:

  1. HTML – структура сайта.
  2. CSS – стилизация сайта.
  3. JavaScript (опционально) – для интерактивных элементов.

Вот пример очень простой веб-страницы:

1. HTML (структура)

html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Простой сайт</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Добро пожаловать на мой сайт!</h1>
    </header>

    <section>
        <h2>Обо мне</h2>
        <p>Привет! Это простой сайт, созданный для демонстрации.</p>
    </section>

    <footer>
        <p>&copy; 2024 Мой сайт</p>
    </footer>
</body>
</html>

2. CSS (стилизация)

css
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
    color: #333;
}

header {
    background-color: #4CAF50;
    color: white;
    padding: 10px 0;
    text-align: center;
}

section {
    margin: 20px;
    padding: 20px;
    background-color: white;
    border-radius: 5px;
    box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1);
}

footer {
    text-align: center;
    padding: 10px;
    background-color: #4CAF50;
    color: white;
    position: fixed;
    bottom: 0;
    width: 100%;
}

3. JavaScript (добавление интерактивности, например, кнопка с приветствием)

html
<button onclick="sayHello()">Нажми меня</button>

<script>
function sayHello() {
    alert('Привет, пользователь!');
}
</script>

Теперь, если у вас есть какой-то конкретный контент или дополнительные пожелания, можно адаптировать этот шаблон под ваши нужды. Сайт можно развернуть на любом хостинге, например, GitHub Pages, если нужно бесплатное решение.

Описание и шаги:

  1. Структура страницы: В HTML добавлены разделы "Про компанію", "Вакансії" и "Контакти".
  2. Стилизация: В CSS создана аккуратная и современная стилизация, с чистым дизайном и плавными элементами.
  3. Навигация: В верхней части сайта добавлено меню для быстрого перехода между разделами.

Как разместить сайт:

  • Можете загрузить файлы на любой хостинг (например, GitHub Pages или Netlify).
  • Файл HTML сохраните как index.html, а CSS – как style.css в одной директории.
